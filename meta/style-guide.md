# Style Guide

## File Naming

- Chapters: `01-chapter-title.md`
- Interludes: `05-interlude-title.md`
- Appendices: `90-appendix-title.md`

## Markdown Conventions

- Use one `#` heading for the chapter title
- Leave a blank line between paragraphs
- Avoid inline planning notes in manuscript files
- Put revision comments in `notes/`, not in chapter drafts
